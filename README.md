###项目介绍
选择相册图片，添加多行文字并进行个性化排版样式设计，最后再保存成图片的开发学习项目。

###项目学习内容
- MVP结构实践
- EventBus的开源库使用
- Picasso图片加载开源库的简单使用
- 自定义多层绘制控件的开发
- 相册图片加载及保存图片到相册实时更新功能实践
- Material Design界面控件（RecyclerView/CardView/CoordinatorLayout/FloatingActionButton)的学习实践
- 自定义DependentBehavior学习实践
- 自定义程序崩溃信息生成日志文件功能实践
